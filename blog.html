Directives

AngularJS directives are HTML attributes with an ng prefix. There are many different ng prefixes including ng-ap which tells angular js that the div element is the owner of the AngularJS application. The ng-model directive binds the value of the input field to the application variable name and the ng-bind directive binds the innerHTML of the p element to the application variable name. These are just a view of the directives there are many more.

Take a look at these 4 examples below

The ng-app Directive

The ng-app directive will auto-bootstrap the application when a web page is loaded. It can also have a value to connect to code modules!

The ng-init Directive

The ng-init directive can define initial values for your application. You will normally use a controller of module instead.

The ng-model Directive

The ng-model directive binds the value of HTML controls to application data. It can also provide validation, CSS classes, and status for applicaton data!

Controllers

AngularJS applications are controlled by controllers. The ng-controller directive defines the application controller. A controller is a JavaScript Object, created by a standard JavaScript object constructor.

Take a look at these 2 examples below

Example Controller File

function namesController($scope) {
$scope.names = [
{name:'Jani',country:'Norway'},
{name:'Hege',country:'Sweden'},
{name:'Kai',country:'Denmark'}
]; }

Call Controller File in an App

div ng-app"" ng-controller="namesController"
li ng-repeat="x in names"
{{x.name+', ' + x.country}}
/li
/div
script src="namesController.js" /script

Expressions

AngularJS expressions are written inside double braces: {{ expression }}. AngularJS expressions binds data to HTML the same way as the ng-bind directive. AngularJS will "output" data exactly where the expression is written.

Data Binding

The {{ firstName }} expression, in the example above, is an AngularJS data binding expression. Data binding in AngularJS, synchronizes AngularJS expressions with AngularJS data. {{ firstName }} is synchronized with ng-model="firstName". In the next example two text fields are synchronized with two ng-model directives.
